/* PDFolio | Dijital Dergi Dönüştürücü */

PDFolio, standart PDF dosyalarınızı tarayıcı üzerinde çalışan, yüksek kaliteli, animasyonlu ve interaktif bir dijital dergiye dönüştüren modern bir web uygulamasıdır.

* Özellikler :

  -> Akıllı Performans Yönetimi

  -> Yakınlaştırma / Uzaklaştırma (Zoom)

  -> Sayfa Navigasyonu (İleri/Geri)

  -> Sürükle ve Bırak dosya yükleme desteği

  -> Lazy Loading

  -> Bellek Koruması

  -> Mobil uyumlu ve modern tasarım

* Kullanılan Teknolojiler :

  -> HTML / CSS
  -> JavaScript
  -> PDF.js

/* PDFolio | Digital Magazine Converter */

PDFolio is a modern web application that turns your standard PDF files into a high-quality, animated and interactive digital magazine that runs in the browser.

* Features: 

  -> Intelligent Performance Management 

  -> Zoom In / Zoom Out (Zoom) 

  -> Page Navigation (Forward/Back) 

  -> Drag and Drop file upload support 

  -> Lazy Loading 

  -> Memory Protection 

  -> Mobile compatible and modern design

* Technologies Used:

  -> HTML/CSS
  -> JavaScript
  -> PDF.js

* Developer: Eren Söğütlü

LINK: https://erensogutlu-pdfolio.netlify.app/
